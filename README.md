# FocoNauta

Projeto desenvolvido pelos alunos de análise e desenvolvimento de sistemas do 5º semestre da Faculdade Senac DF, para validação de conhecimentos adquiridos na disciplinas cursadas.
Desenvolvimento desenvolvimento do projeto FocoNauta, um sistema responsivo de aprendizado baseado em quizzes. O projeto visa proporcionar uma experiência educacional envolvente e eficaz para os usuários.
